# Pokemon Shiritori

## How to dev

### Setup

```bash
# ... Setup node and yarn
$ git clone git@github.com:kobakazu0429/pokemon-shiritori.git
$ cd <your project name>
$ yarn install
```

### Commands

```bash
# written in package.json
$ yarn dev   # Start webpack-dev-server
$ yarn build # Build to dist/
$ yarn lint  # Check type and syntax
```
